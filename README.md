 Hospital Emergency Room Dashboard

A Power BI dashboard designed to analyze and monitor key performance indicators (KPIs) of a hospital’s Emergency Room (ER). This dashboard helps hospital administrators, doctors, and analysts understand patient flow, staff workload, treatment times, and overall ER efficiency.

 Project Overview

The Hospital Emergency Room Dashboard provides a detailed view of:

Total ER visits

Patient arrival trends

Waiting times

Treatment durations

Admission vs. discharge rate

Patient categories (age, gender, criticality level)

Staff performance

Peak hours and daily load distribution

The dashboard helps improve decision-making, optimize resource allocation, and enhance patient care quality.

 Dataset

The dataset includes (based on typical ER data):

Patient demographics

Check-in time

Waiting time

Treatment time

Department

Severity level

Doctor/Nurse assignment

Admission status
